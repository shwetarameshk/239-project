Decision Tree

R is a programming language that is used for data manipulation and analysis, statistical computing and generating graphs. There are many algorithms and techniques that are available in the form of packages. These packages can be used to analyze the data set. We have written a program in R, using RStudio as the IDE, to generate a robust decision tree that accurately classifies a given set of attributes for breast cancer prediction.

We used the ‘rpart’ package, an efficient classification algorithm, to generate the decision tree. The ‘rpart’ package provides recursive partitioning that helps build classification and regression trees.
